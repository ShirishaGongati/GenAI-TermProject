## Project Summary

This project demonstrates how Large Language Models (LLMs), including GPT-4o and Llama3.1, can automatically generate structured manual test cases from natural-language requirements.  

The system is implemented in a Jupyter Notebook with:
- A text box for entering requirements  
- A **Generate Test Cases** button  
- An output area showing the LLM-generated test cases  
- Support for GPT-4o and local Llama3.1 (Ollama)  
- Basic error handling and fallback
